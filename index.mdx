Letta Open ADE (Agent Development Environment) gives you a complete operator workspace for Letta-powered AI agents. From a single interface you can create agents, edit their system prompts and personas, attach tools, chat with them in real time, and run automated validation suites — all backed by a versioned REST API.

## How ADE works

<Steps>
  <Step title="Deploy the stack">
    Start the Letta server, Postgres, Redis, and ADE services with a single `docker compose up -d`.
  </Step>
  <Step title="Connect an LLM backend">
    Point ADE at Doubao (Ark), LM Studio, or any OpenAI-compatible endpoint via environment variables.
  </Step>
  <Step title="Create your first agent">
    Use Agent Studio or the REST API to create a named agent with your chosen model, embedding, and system prompt.
  </Step>
  <Step title="Iterate with confidence">
    Edit prompts and personas, attach tools, run test suites, and review revision history — all without redeploying.
  </Step>
</Steps>
